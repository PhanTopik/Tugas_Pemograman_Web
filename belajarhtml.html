Mengenal HTML, CSS, Javascript dan Tailwind

HTML (Hypertext Markup Language)

HTML adalah bahasa markup standar untuk membuat dan menata konten halaman web. HTML berfungsi sebagai kerangka dasar untuk website, menggunakan tag-tag untuk mendefinisikan struktur seperti judul, paragraf, gambar, dan tautan.

CSS (Cascading Style Sheets)

CSS adalah adalah bahasa stylesheet yang digunakan untuk mendeskripsikan tampilan dokumen, seperti situs web, yang ditulis dalam bahasa markup seperti HTML atau XML ibaratkan kita yang memakai baju yang bagus agar terlihat rapi dan bagus.

JavaScript

bahasa pemrograman yang digunakan untuk membuat halaman web interaktif dan dinamis, memungkinkan fitur seperti animasi, pembaruan konten secara real-time, dan interaksi pengguna lainnya. Bisa kita ibaratkan sebagai Sistem Syaraf dalam tubuh kita yang bertugas membuat interaksi dan gerakan pada kita.

Tailwind

Tailwind adalah framework CSS utility-first yang memungkinkan pengembang membuat desain kustom langsung di markup HTML menggunakan kelas-kelas utilitas yang sudah ditentukan sebelumnya

Cara Membuat Website Sederhana

  1. Download aplikasi VS code dan ekstension HTML ,CSS ,Javascript ,dan Tailwind untuk membuat web
  2. buat folder untuk menampung program website kamu yang berisi file index.html ,style.css ,script.js ,dan folder img agar tidak berantakan
  3. lalu selanjutnya maasukan kode untuk membuat website
                
    <!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Sederhana</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <header>
    <h1>Website Sederhana Saya</h1>
  </header>

  <main>
    <h2>Selamat Datang!</h2>
    <p>Ini contoh website sederhana menggunakan HTML, CSS, dan JavaScript.</p>

    <div class="card">
      <h3>Kartu Informasi</h3>
      <p>Klik tombol di bawah untuk interaksi sederhana.</p>
      <button id="klikTombol">Klik Saya</button>
    </div>
  </main>

  <footer>
    © 2025 Contoh Website
  </footer>

  <script src="script.js"></script>
</body>
</html>

List Tag dasar yang wajib ada

  • !DOCTYPE html: ini berfungsi sebagai deklarasi atau instruksi untuk browser
  • html: ini berfungsi sebagai tag pembuka dari keseluruhan dokumen HTML
  • head: Bagian ini berisi metadata atau informasi tentang dokumen HTML, bukan konten yang akan ditampilkan di halaman.
  • meta charset="UTF-8": untuk Menentukan set karakter yang digunakan.
  • meta name="viewport" content="width=device-width, initial-scale=1.0": untuk desain responsif.
  • title: untuk Menentukan judul halaman yang akan muncul di tab browser, bookmark, dan hasil pencarian.
  • link rel="stylesheet" href="style.css": untuk Menghubungkan file HTML dengan file CSS eksternal untuk mengatur gaya (tampilan) halaman (Hanya digunakan jika mau menghubungkan dengan file eksternal).
  • Body: bagian yang berisi semua konten yang terlihat oleh pengunjung halaman web.
